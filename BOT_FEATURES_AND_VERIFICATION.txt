# JINBE DISCORD BOT - FEATURES & CAPABILITIES

## What This Bot Does

Jinbe is a powerful Discord bot designed for server automation, moderation, and management with advanced template systems.

# Core Features

. Server Template System
  - 5 pre-built server templates (Gaming, Music, Friends, Blox Fruits, YouTube)
  - Automatic channel and role creation
  - Category organization
  - Custom permissions setup
  - Template application deletes all existing channels and rebuilds from scratch

. Advanced Auto-Moderation
  - Profanity filter with bypass pattern detection
  - Leet speak detection
  - Progressive warning system (3 warnings = 1 hour ban, 5 warnings = kick)
  - NSFW content detection in non-NSFW channels
  - Anti-spam system (5 messages in 10 seconds triggers action)
  - Automatic temporary bans with scheduled unbans
  - Staff channel logging for all moderation actions

. Channel Management
  - Lock channels to prevent member messages
  - Unlock previously locked channels
  - Permission-based access control
  - Owner-only command restrictions

. Announcement System
  - Formatted announcement embeds
  - @everyone ping capability
  - Automatic channel creation if not exists
  - Template-aware channel naming

. Welcome System
  - Custom welcome messages with member mentions
  - Automatic role assignment based on template
  - Welcome embeds with member information
  - Member count tracking
  - Account age display

. Quote System
  - Create beautiful quote embeds from message links
  - Context menu integration (right-click ‚Üí Create Quote)
  - Attachment preview support
  - Reaction count display
  - Jump link to original message
  - Optional reply text

. Rules Management
  - Auto-generated rules based on server template
  - Custom rule editing
  - Rules channel with permissions (owner + bot only)
  - Template-specific rule sets

. Data Persistence
  - Stores warning history in bot_data.json
  - Saves welcome messages per server
  - Tracks banned users and unban times
  - Server configuration storage
  - Automatic data backup

. Blox Fruits Crew Features
  - Automatic bounty role system (5M, 10M, 15M, 20M, 30M)
  - Marine role system
  - Crew name role creation
  - Fleet Admiral hierarchy

. YouTube Community Features
  - Subscriber milestone roles (1K to 1M+)
  - Milestone tracking channels
  - Content creator role management
  - Achievement celebrations

# Commands

## Slash Commands (Owner Only)

. /templates - View available server templates with interactive menu
. /apply <template> - Apply a template (WARNING: Deletes all existing channels)
. /announce <message> - Make server announcements with embeds
. /welcome <message> - Set custom welcome message (use {member} for mention)
. /editrules <rules> - Edit server rules (one per line)
. /sync - Manually sync commands if not showing
. /global_sync - Force global command sync (Bot owner only)
. /help - Show comprehensive help menu

## Prefix Commands (Owner Only)

. !lock <#channel> - Lock a channel to prevent messages (defaults to current channel)
. !unlock <#channel> - Unlock a locked channel (defaults to current channel)

## Public Commands

. /quote <message_link> [reply_text] - Create quote from message link
. Right-click message ‚Üí Apps ‚Üí Create Quote - Quick quote creation
